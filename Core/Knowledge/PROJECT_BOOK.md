# NASRIUM Project Book

## 1) چشم‌انداز
ساخت یک اکوسیستم جهانی بازی + اقتصاد دیجیتال که از تلگرام شروع می‌شود و به بلاکچین/محصولات چندپلتفرمی توسعه می‌یابد.

## 2) اصول
- کیفیت بالاتر از سرعت
- امنیت بالاتر از امکانات
- مستندسازی همزمان با توسعه
- هر تغییر باید دلیل و ثبت تصمیم داشته باشد (ADR)
